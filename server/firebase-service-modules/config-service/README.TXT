This service provides configuration for the system.

The configuration can be changed by modifying the properties specified in the @Configurated classes in the server.props??? file.

For example, the defaults in PokerActivatorConfig can be modified by changing (or adding) the following properties:

com.cubeia.game.poker.activator.use-mock-integrations
com.cubeia.game.poker.activator.default-table-ttl
com.cubeia.game.poker.activator.get-activator-interval
